About CubicTest

CubicTest has been in the works since spring 2005 and has been actively developed since then. CubicTest started as a research project at Bekk Consulting AS to facilitate functional test driven development and automated web acceptance testing. CubicTest is now in a stable release. Four master's theses have contributed to CubicTest, as well as continuous work by the project lead.
