Upgrading the Update Site version

Use the standard Eclipse plug-in upgrade mechanism to upgrade CubicTest:
  • In Eclipse select Help -> Software Updates -> Manage Configuration.
  • Locate CubicTest (expand Eclipse), right click on it and select Find Updates....
  • Complete the standard Eclipse plugin upgrade process.
  • When Eclipse is restarted, you need to Reset the CubicTest Perspective to get shortcuts to the new features. To reset, open the CubicTest Perspective and select Window -> Reset Perspective.
If you run tests from JUnit...
  • Edit your test project's pom.xml file and update the version of the cubictest dependencies to the latest version.
  • Open up a command shell and run the maven command mvn eclipse:eclipse to update the CubicTest jars used by JUnit in Eclipse to the latest version.

Upgrading the Rich Client version

To upgrade the Rich Client version,
  • Download and extract the new CubicTest version as normal.
  • Copy the CubicTest / Eclipse "workspace" directory to the new version of CubicTest.
  • When CubicTest is started, you need to Reset the Perspective to get shortcuts to the new features . In CubicTest, select Window -> Reset Perspective.
